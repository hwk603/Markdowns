###正则匹配

#### 题目：
```
1. 已01结尾的01字符串；
2. 能被5整除的正整数；
3. 不含abb的ab字符串；
4. 含有010的01字符串；
5. 每个0后面都有一个1的01字符串；
```
<!--more-->

#### 答案：
```
1. (0|1)*01
2. (([1-9][0-9]*|ε)5) | (([1-9][0-9]*)0) | 5
3. b*(a|ba)*(b|ε)
4. (0|1)*010(0|1)*
5. (1|01)*
```
